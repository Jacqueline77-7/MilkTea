# 今日茶室 · 心情奶茶推荐

纯前端移动端网页：做一份小测试，根据心情推荐「今日幸运茶」。无后端、无构建步骤，所有数据内联在 `js/` 中。

## 部署到 GitHub Pages（三步）

1. 在 GitHub 新建一个仓库（例如 `mood-tea`）。
2. 把本目录下的文件（`index.html`、`styles.css`、`js/`、`assets/`）上传到仓库**根目录**。
3. 进入仓库 **Settings → Pages**，Source 选 **Deploy from a branch**，Branch 选 `main`（或 `master`），目录选 **/ (root)**，保存。

约 1–2 分钟后即可访问：`https://<你的用户名>.github.io/<仓库名>/`

> 纯静态站点，上传即用，无需任何构建或安装。
